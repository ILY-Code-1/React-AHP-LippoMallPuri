White Box Testing

Aplikasi Enerlyze — Smart Energy Analysis & AHP Decision Engine · Lippo Mall Puri
Pengujian struktural berbasis flowchart, flowgraph, cyclomatic complexity, dan basis path.

Pengujian white box difokuskan pada modul-modul inti yang memuat percabangan logika non-trivial — yaitu fungsi autentikasi, penyimpanan data dengan guard duplikasi, mesin AHP, dan pengaturan rank kriteria. Rumus yang dipakai: V(G) = E − N + 2P (E = jumlah edge, N = jumlah node, P = jumlah komponen terhubung) yang ekuivalen dengan V(G) = jumlah predikat + 1.

Fungsi 3 — buildCriteriaMatrix()

Code traceability: berkas src/lib/ahp.js baris 27–43 (pembentukan matriks pairwise Saaty dari rank kriteria).

3.1 Flowchart

@startuml
title Flowchart - buildCriteriaMatrix()
start
:Inisialisasi matrix n x n bernilai 1; i = 0;
while (i < n ?) is (ya)
  :j = 0;
  while (j < n ?) is (ya)
    if (i sama dengan j ?) then (ya)
      :matrix[i][j] = 1;
    else (tidak)
      :Hitung diff rank & nilai Saaty;
      if (diff >= 0 ?) then (ya)
        :matrix[i][j] = saaty;
      else (tidak)
        :matrix[i][j] = 1 / saaty;
      endif
    endif
    :j = j + 1;
  endwhile (tidak)
  :i = i + 1;
endwhile (tidak)
:Kembalikan matrix;
stop
@enduml
