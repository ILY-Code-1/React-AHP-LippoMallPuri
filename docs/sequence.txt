' == Sequence: Login ==
@startuml
title Sequence Diagram - Login Enerlyze
actor "Pengguna" as U
participant "Halaman Login" as UI
participant "authService" as SVC
database "Firestore" as DB

U -> UI : Masukkan email & password
activate UI
UI -> SVC : login(email, password)
activate SVC
SVC -> DB : Ambil user berdasarkan email
activate DB
DB --> SVC : Data user
deactivate DB
alt Kredensial valid
  SVC --> UI : Data user terverifikasi
  UI -> UI : Buat sesi & arahkan ke dashboard
  UI --> U : Tampilkan dashboard sesuai role
else Kredensial salah
  SVC --> UI : Error "Email atau password salah"
  UI --> U : Tampilkan pesan kesalahan
end
deactivate SVC
deactivate UI
@enduml
