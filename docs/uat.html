User Acceptance Testing (UAT)

Aplikasi Enerlyze — Smart Energy Analysis & AHP Decision Engine · Lippo Mall Puri
Pengujian penerimaan dari sudut pandang pengguna akhir. Role penguji: Admin & Staff — sesuai aktor pada use case diagram.
No Skenario Pengujian Role Fitur yang Diuji Langkah Pengujian Ekspektasi Hasil Status
1 Staff masuk ke sistem untuk mulai bekerja Staff Login Membuka aplikasi, memasukkan email & password staff, menekan tombol Login. Berhasil masuk dan diarahkan ke Dashboard Staff. Pass
2 Admin masuk ke sistem untuk mengelola data Admin Login Membuka aplikasi, memasukkan email & password admin, menekan tombol Login. Berhasil masuk dan diarahkan ke Dashboard Admin. Pass
3 Pengguna mengganti password demi keamanan akun Staff Ubah Password Membuka menu Ubah Password, mengisi password lama & password baru, menekan Simpan. Muncul notifikasi password berhasil diperbarui dan dapat dipakai login berikutnya. Pass
4 Staff memantau ringkasan konsumsi energi areanya Staff Lihat Dashboard Membuka halaman Dashboard, memilih rentang periode pada dropdown, lalu mengklik salah satu bar/titik area pada grafik. Kartu KPI & grafik skor AHP tampil sesuai periode (mulai bulan sebelumnya); klik bar/titik menampilkan panel detail skor, ranking & data input area tsb. Pass
5 Staff menginput data energi bulanan areanya Staff Kelola Data Energi Membuka halaman Data, memilih area & bulan, klik Tambah Data, mengisi kWh, biaya, durasi, maintenance, lalu Simpan. Data energi tersimpan dan tampil pada tabel area & bulan terkait. Pass
6 Staff memperbaiki data energi yang keliru Staff Kelola Data Energi Memilih data yang sudah ada, klik ikon Edit, mengubah nilai, lalu Simpan. Nilai data energi diperbarui sesuai input terbaru. Pass
7 Admin menjalankan analisis prioritas optimasi energi Admin Jalankan Analisis AHP Membuka Data Admin, memilih bulan dengan data lengkap, klik Run AHP, mengonfirmasi. Sistem menghitung & menampilkan ranking prioritas area beserta skor AHP. Pass
8 Admin meninjau laporan detail hasil AHP Admin Lihat Laporan Detail AHP Pada Data Admin, klik View Detail untuk bulan terpilih. Laporan tampil lengkap: ranking, pairwise matrix, kontribusi kriteria, & rekomendasi tindakan. Pass
9 Admin mengunduh laporan AHP untuk arsip/rapat Admin Unduh Laporan PDF Pada halaman laporan, klik tombol Unduh PDF. Berkas PDF laporan AHP terunduh ke perangkat dengan tampilan rapi. Pass
10 Admin menghapus hasil AHP yang tidak terpakai Admin Hapus Hasil AHP Pada halaman laporan, klik Hapus Laporan lalu konfirmasi. Hasil AHP terhapus, sedangkan data input energi tetap tersimpan. Pass
11 Admin menambah akun pengguna baru Admin Kelola Master User Membuka Master User, klik Tambah User, mengisi nama, email, role, password, lalu Simpan. Akun pengguna baru tersimpan & tampil pada daftar user. Pass
12 Admin mereset password pengguna yang lupa Admin Kelola Master User Pada Master User, klik ikon Reset Password pada user terkait, lalu konfirmasi. Password user direset ke nilai default dan ditampilkan ke admin. Pass
13 Admin mendaftarkan area gedung beserta penanggung jawab Admin Kelola Master Area Membuka Master Area, klik Tambah Area, mengisi nama & deskripsi, memilih user PJ, lalu Simpan. Area baru tersimpan dengan satu user penanggung jawab. Pass
14 Admin menyesuaikan prioritas kriteria penilaian Admin Kelola Kriteria AHP Membuka Master AHP, klik Edit pada kriteria, memilih rank baru, lalu Simpan. Rank kriteria diperbarui & rank kriteria lain tertukar otomatis agar tetap unik. Pass
15 Staff memeriksa notifikasi pengingat input data Staff Kelola Notifikasi Klik ikon lonceng, membaca daftar notifikasi, mengklik salah satu notifikasi. Daftar notifikasi tampil & klik mengarahkan ke halaman input data terkait. Pass
16 Admin membersihkan seluruh notifikasi Admin Kelola Notifikasi Klik ikon lonceng, klik "Tandai semua terbaca". Seluruh notifikasi terhapus dan indikator lonceng menjadi kosong. Pass
Kesimpulan UAT: 16 skenario penerimaan yang mencakup seluruh 12 fitur Enerlyze dan diuji oleh role Admin serta Staff seluruhnya berstatus Pass. Aplikasi dinyatakan dapat diterima dan layak digunakan oleh pengguna akhir di lingkungan Lippo Mall Puri.
